# IA - Proyecto Académico Empresarial

Bienvenido al repositorio del proyecto académico **IA**, donde exploraremos conceptos de Inteligencia Artificial mediante cursos prácticos implementados con Node.js.

## 📚 Descripción

Este proyecto es una colección de cursos y recursos académicos sobre Inteligencia Artificial, desarrollados con **Node.js** como plataforma principal de aprendizaje.

## 🎯 Objetivos

- Aprender conceptos fundamentales de IA
- Implementar algoritmos y modelos prácticos
- Desarrollar habilidades en desarrollo con Node.js
- Compartir conocimiento académico

## 📂 Estructura del Proyecto

```
IA/
├── cursos/                 # Directorios de cursos
│   ├── curso-1/
│   ├── curso-2/
│   └── ...
├── src/                    # Código fuente general
├── resources/              # Recursos y documentación
├── tests/                  # Pruebas unitarias
├── package.json           # Dependencias Node.js
└── README.md             # Este archivo
```

## 🚀 Primeros Pasos

### Requisitos
- Node.js (v14 o superior)
- npm o yarn

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/jotaprado/IA.git
cd IA

# Instalar dependencias
npm install
```

### Ejecutar un Curso

```bash
npm run start
```
